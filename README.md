# Provider Counter App

A Flutter app demonstrating state management using Provider.

## Features
- Counter increment & decrement
- State management using Provider
- Clean and scalable architecture

## What is Provider?
Provider is a state management solution in Flutter that helps manage and share data efficiently across the app.

## Technologies
- Flutter
- Dart
- Provider

## How to Run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
